# 🎓 Student Grade Analyzer

A Python command-line tool that analyzes student marks, calculates grades, generates visual bar charts and produces a complete class report.

## ✨ Features

- Add multiple students with multiple subjects
- Auto calculates average, grade and remarks
- Visual bar chart for each student in terminal
- Shows class topper and weakest student
- Pass/fail count for entire class
- Demo mode with pre-loaded sample data

## 🛠️ Tech Stack

- Python 3
- Statistics module (built-in)
- OS module (built-in)
- No external libraries needed!

## ▶️ How to Run
```bash
python student_grade_analyzer.py
```

## 📊 Grade Scale

| Marks | Grade | Remark |
|-------|-------|--------|
| 90-100 | A+ | Outstanding |
| 80-89 | A | Excellent |
| 70-79 | B | Good |
| 60-69 | C | Average |
| 50-59 | D | Below Average |
| 0-49 | F | Fail |
